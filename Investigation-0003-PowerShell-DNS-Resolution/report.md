# Investigation ID: INV-0003

## Title

PowerShell DNS Resolution Investigation

---

## Date

2026-07-23

---

## Lab Environment

- **SIEM:** Elastic Security
- **Endpoint:** Windows 10
- **Log Source:** Sysmon & PowerShell Operational Logs
- **Host:** DESKTOP-0DFKSAT

---

## Objective

Determine whether the observed DNS queries executed through PowerShell represent normal user activity or suspicious behavior.

---

## Summary

User **gamal** on **DESKTOP-0DFKSAT** launched **powershell.exe** from **explorer.exe**. During the session, the user executed the **Resolve-DnsName** PowerShell cmdlet to resolve **google.com** and **github.com**. The DNS requests were successfully completed and returned valid public IP addresses. No suspicious behavior or malicious indicators were identified during this investigation.

## Investigation

The investigation began by identifying the creation of **powershell.exe** using **Sysmon Event ID 1**.

The parent process was **explorer.exe**, indicating that PowerShell was launched interactively by the logged-in user.

A pivot using the process ID (**9548**) and event timestamps led to the corresponding **PowerShell Operational** logs. These logs confirmed that the **Resolve-DnsName** cmdlet was executed twice.

Correlation with **Sysmon Event ID 22** confirmed successful DNS resolution for both **google.com** and **github.com**. The returned IP addresses matched legitimate public infrastructure associated with Google and GitHub.

No encoded PowerShell commands, obfuscated scripts, suspicious child processes, or indicators of malicious activity were observed during the investigation.

---

## Analysis

The observed activity is consistent with normal interactive PowerShell usage.

The executed **Resolve-DnsName** cmdlets generated legitimate DNS requests for public domains.

The parent-child relationship (**explorer.exe → powershell.exe**) indicates user-initiated execution.

DNS telemetry confirmed successful name resolution without any indicators of command obfuscation, suspicious domains, or malicious behavior.

No evidence of persistence, privilege escalation, defense evasion, or unauthorized execution was identified within the scope of this investigation.
